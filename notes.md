# TGC Development Notes

## Latest Updates

### Theme Simplification - Pure MUI with Color Overrides Only

**Date:** November 8, 2025

**IMPORTANT CHANGE:** Reverted to minimal theme configuration - ONLY colors are overridden, everything else uses MUI defaults.

**New Theme Structure:**

**Light Theme:**
```js
palette: {
  mode: 'light',
  primary: { main: '#6366F1' },      // Indigo
  secondary: { main: '#EC4899' },    // Pink
  warning: { main: '#F59E0B' },      // Amber
  success: { main: '#10B981' },      // Green (was 'accent')
}
```

**Dark Theme:**
```js
palette: {
  mode: 'dark',
  primary: { main: '#818CF8' },      // Light Indigo
  secondary: { main: '#F472B6' },    // Light Pink
  warning: { main: '#FBBF24' },      // Bright Amber
  success: { main: '#34D399' },      // Bright Green (was 'accent')
}
```

**What's NOT Overridden (Uses MUI Defaults):**
- ❌ Typography scales
- ❌ Spacing system
- ❌ Shadows/elevation
- ❌ Border radius
- ❌ Background colors (auto-generated by MUI)
- ❌ Text colors (auto-generated by MUI)
- ❌ Component styles (except textTransform: 'none' for buttons)

**Benefits:**
- ✅ Pure Material Design system
- ✅ MUI handles all responsive behavior
- ✅ MUI handles accessibility automatically
- ✅ MUI generates proper color variants (light, dark, contrastText)
- ✅ Theme adapts to platform conventions
- ✅ Less code to maintain
- ✅ Future MUI updates work seamlessly

**Note:** Used `success` instead of `accent` as MUI's standard color name for green.

## Recent Improvements

### 1. Mobile-Responsive Payout Screen
**Issue:** Table was not responsive on mobile devices  
**Solution:** Implemented adaptive layout:
- Desktop (>960px): Full table with all columns
- Mobile (<960px): Card-based layout with:
  - Event name and host at top
  - Status chip in corner
  - Amount and date in two columns
  - Full-width action button at bottom

**Benefits:**
- ✅ Touch-friendly on mobile
- ✅ All information visible
- ✅ Better UX on small screens
